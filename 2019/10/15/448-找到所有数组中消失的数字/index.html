448. 找到所有数组中消失的数字

Posted on 2019-10-15 In Leetcode Views: Disqus:
LeetCode刷题系列。

描述

给定一个范围在  1 ≤ a[i] ≤ n ( n = 数组大小 ) 的 整型数组，数组中的元素一些出现了两次，另一些只出现一次。

找到所有在 [1, n] 范围之间没有出现在数组中的数字。

您能在不使用额外空间且时间复杂度为O(n)的情况下完成这个任务吗? 你可以假定返回的数组不算在额外空间内。

解题思路

本题的要求比较多，不过也给足了相关条件，特别是这个整型数组的范围为 1 ≤ a[i] ≤ n，对此，我们的思路是如果整个数组是有序的，nums[i]元素正好都在其对应的位置nums[nums[i] - 1]上面，比如对于没有缺失项的正确的顺序应该是[1, 2, 3, 4, 5, 6, 7, 8]，而我们现在却是[4,3,2,7,8,2,3,1]，我们需要把数字移动到正确的位置上去，比如第一个4就应该和7先交换个位置，以此类推，最后得到的顺序应该是[1, 2, 3, 4, 3, 2, 7, 8]，我们最后在对应位置检验，如果nums[i]和i+1不等，那么我们将i+1存入结果res中即可，具体代码如下：
