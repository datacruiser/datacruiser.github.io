Leetcode Tencent 50 Task 29 217. Contains Duplicate

DataWhale暑期学习小组-LeetCode刷题第八期Task29。

描述

给定一个整数数组，判断是否存在重复元素。

如果任何值在数组中出现至少两次，函数返回 true。如果数组中每个元素都不相同，则返回 false。

解题思路

想到了两个解法，第一个解法需要遍历两遍数组，如果找到相等的元素返回true, 否则, 返回false, 时间复杂度为O(N2)O(N^2)O(N2).
但是这个解法会有超时.

第二个解法是先排序, 然后后面只需要遍历一遍数组就可以了, 比较相邻元素, 如果有重复的返回true, 否则, 返回false.
